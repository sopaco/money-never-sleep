## 移动平均线

### 简单移动平均线 (SMA)

**定义**: n 周期内价格的平均值

**计算公式**:
```
SMA(n) = (P1 + P2 + ... + Pn) / n
```

**常用周期**:
- SMA 20: 短期趋势
- SMA 50: 中期趋势
- SMA 200: 长期趋势

**解读**:
- 价格在 SMA 上方 → 上升趋势
- 价格在 SMA 下方 → 下降趋势
- SMA 交叉 → 趋势反转信号

### 指数移动平均线 (EMA)

**定义**: 对近期价格赋予更高权重的移动平均

**计算公式**:
```
EMA(t) = α × Price(t) + (1 - α) × EMA(t-1)
其中 α = 2 / (n + 1)
```

**常用周期**:
- EMA 12 / EMA 26: MACD 计算
- EMA 9: MACD 信号线
- EMA 21: 短期趋势

**优势**:
- 对价格变化更敏感
- 更快反映趋势变化

## 相对强弱指数

### 定义

RSI 是衡量价格变动速度和变化幅度的动量振荡指标。

### 计算公式

```
RSI = 100 - (100 / (1 + RS))

其中:
RS = 平均涨幅 / 平均跌幅
```

**详细步骤**:
1. 计算每个周期的价格变化
2. 分离上涨和下跌
3. 计算平均上涨和平均下跌（通常使用 14 周期）
4. 计算 RS = 平均上涨 / 平均下跌
5. 应用公式得出 RSI

### 参数

- **默认周期**: 14
- **常用变体**: RSI 7 (更敏感), RSI 21 (更平滑)

### 解读指南

| RSI 值 | 状态 | 操作提示 |
|--------|------|----------|
| > 70 | 超买 | 潜在卖出信号 |
| < 30 | 超卖 | 潜在买入信号 |
| 50 - 70 | 偏强 | 上升趋势 |
| 30 - 50 | 偏弱 | 下降趋势 |

### 高级用法

**背离信号**:
- 看涨背离: 价格创新低，RSI 未创新低 → 反弹可能
- 看跌背离: 价格创新高，RSI 未创新高 → 回调可能

**中轴线交叉**:
- RSI 上穿 50 → 上升趋势确认
- RSI 下穿 50 → 下降趋势确认

## MACD

### 定义

MACD (Moving Average Convergence Divergence) 是趋势跟踪动量指标。

### 组成部分

1. **MACD 线**: EMA(12) - EMA(26)
2. **信号线**: MACD 线的 EMA(9)
3. **柱状图**: MACD 线 - 信号线

### 计算公式

```
MACD 线 = EMA(12) - EMA(26)
信号线 = EMA(MACD, 9)
柱状图 = MACD 线 - 信号线
```

### 解读指南

**交叉信号**:
- MACD 上穿信号线 → 买入信号（金叉）
- MACD 下穿信号线 → 卖出信号（死叉）

**零轴线**:
- MACD > 0 → 短期均线在长期上方（上升动能）
- MACD < 0 → 短期均线在长期下方（下降动能）

**背离**:
- 看涨背离: 价格新低，MACD 未新低
- 看跌背离: 价格新高，MACD 未新高

## 布林带

### 定义

由三条轨道组成的波动性指标，反映价格相对于前期交易的范围。

### 组成部分

1. **中轨**: 20 周期 SMA
2. **上轨**: 中轨 + 2 × 标准差
3. **下轨**: 中轨 - 2 × 标准差

### 计算公式

```
中轨 = SMA(20)
上轨 = 中轨 + 2 × σ
下轨 = 中轨 - 2 × σ

其中 σ = 20 周期价格标准差
```

### 解读指南

**带宽**:
- 带宽扩大 → 波动性增加
- 带宽收窄 → 波动性减少（可能突破）

**价格位置**:
- 价格触及上轨 → 相对高位
- 价格触及下轨 → 相对低位
- 价格穿越轨道 → 极端情况

**布林挤压**:
- 带宽极度收窄 → 可能出现大幅突破
- 结合成交量确认突破方向

## 支撑位与阻力位

### 定义

- **支撑位**: 价格下跌时可能获得支撑的价格水平
- **阻力位**: 价格上涨时可能遇到阻力的价格水平

### 计算方法

**1. 历史价格极值**:
```
支撑位 = 历史局部最低点
阻力位 = 历史局部最高点
```

**2. 移动平均线**:
```
支撑位 = SMA(50), SMA(200)
阻力位 = 前期高点
```

**3. 斐波那契回撤**:
```
常见回撤位: 23.6%, 38.2%, 50%, 61.8%, 78.6%
```

**4. 枢轴点**:
```
枢轴点(P) = (最高 + 最低 + 收盘) / 3
阻力位1(R1) = 2 × P - 最低
支撑位1(S1) = 2 × P - 最高
```

### 应用规则

| 情况 | 操作 |
|------|------|
| 价格接近支撑位 | 考虑买入 |
| 价格突破支撑位 | 卖出止损 |
| 价格接近阻力位 | 考虑卖出 |
| 价格突破阻力位 | 追涨买入 |

## 成交量指标

### 成交量移动平均 (VMA)

```
VMA(n) = n 周期成交量的平均值
```

**解读**:
- 成交量高于 VMA → 趋势确认
- 成交量低于 VMA → 趋势减弱

### OBV (On-Balance Volume)

```
如果今日收盘 > 昨日收盘:
  OBV(t) = OBV(t-1) + 今日成交量
如果今日收盘 < 昨日收盘:
  OBV(t) = OBV(t-1) - 今日成交量
否则:
  OBV(t) = OBV(t-1)
```

**应用**:
- OBV 上升 + 价格上涨 → 强势
- OBV 下降 + 价格上涨 → 背离（警告信号）

## 波动率指标

### ATR (Average True Range)

**计算**:
```
True Range = max(
  High - Low,
  |High - Previous Close|,
  |Low - Previous Close|
)

ATR(n) = n 周期 True Range 的 SMA
```

**应用**:
- 止损位设置: 入场价 - 2 × ATR
- 仓位管理: 波动性大的资产减少仓位

### 历史波动率

**计算**:
```
1. 计算对数收益率: r = ln(P(t) / P(t-1))
2. 计算收益率标准差: σ
3. 年化波动率 = σ × √252
```

## 注意事项

### 技术指标局限性

1. **滞后性**: 所有技术指标都基于历史数据，存在一定滞后
2. **假信号**: 震荡市场中可能产生频繁假信号
3. **市场环境**: 不同市场条件下指标有效性不同
4. **参数敏感**: 不同参数设置可能产生不同结果

### 最佳实践

1. **多指标结合**: 单一指标不足以做决策
2. **趋势优先**: 在明确趋势中指标更有效
3. **时间框架**: 结合多个时间框架确认信号
4. **风险管理**: 技术分析不保证成功，需配合风控
