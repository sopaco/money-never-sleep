## Data Flows

### Cash balance updates from transactions

- **Source:** transactions
- **Destination:** cash
- **Operations:** UPDATE
- **Triggers:** `buy_position()`, `sell_position()`

### Position updates from buy/sell operations

- **Source:** transactions
- **Destination:** positions
- **Operations:** INSERT, UPDATE, DELETE
- **Triggers:** `buy_position()`, `sell_position()`

### Sentiment snapshot ingestion from external API

- **Source:** finance-query / alternative.me API
- **Destination:** fear_greed_snapshots
- **Operations:** DELETE (by date), INSERT
- **Function:** `save_fear_greed_snapshot()`

### Portfolio valuation and reporting data flow

- **Source:** positions, cash, fear_greed_snapshots
- **Destination:** report generation
- **Operations:** SELECT
- **Function:** `generate_report()`

### Transaction history logging

- **Source:** `buy_position()`, `sell_position()`
- **Destination:** transactions
- **Operations:** INSERT
- **Triggers:** After each position update
