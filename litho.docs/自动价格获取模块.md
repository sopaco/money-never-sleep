# 自动价格获取模块

## 1. 模块概述

### 模块定位

`quote.rs` 是 MNS 系统的基础设施层模块，负责**自动获取资产实时价格**。该模块屏蔽了多数据源的复杂性，为上层模块（如 `report.rs`、`db.rs`）提供统一的价格获取接口。

### 核心价值

| 价值点 | 说明 |
|--------|------|
| **多数据源适配** | 自动识别资产类型，选择最优数据源（天天基金/Yahoo Finance） |
| **统一接口** | 无论基金还是美股，调用方仅需传入代码，无需关心数据源细节 |
| **容错机制** | 单个资产获取失败不影响整体流程，支持部分成功场景 |
| **异步高效** | 基于 `reqwest` 异步客户端，支持高并发请求 |

## 2. 核心数据结构

### PriceUpdate - 价格更新结果

```rust
pub struct PriceUpdate {
    pub asset_code: String,    // 资产代码（如 "020972", "AAPL"）
    pub asset_name: String,    // 资产名称（如 "国泰纳斯达克", "Apple Inc."）
    pub old_price: Option<f64>,// 更新前价格（可能为空）
    pub new_price: f64,        // 更新后价格
    pub source: String,        // 数据源标识（"天天基金" / "Yahoo Finance"）
}
```

**用途**：封装单次价格更新的结果，供 CLI 展示和 DB 持久化使用。

### StockQuote - 完整股票报价

```rust
pub struct StockQuote {
    pub symbol: String,         // 股票代码
    pub name: String,           // 公司名称
    pub price: f64,             // 当前价格
    pub change: f64,            // 涨跌额（绝对值）
    pub change_percent: f64,    // 涨跌幅（百分比）
}
```

**用途**：用于市场分析场景（如 `mns market` 命令），提供完整的报价信息。

## 3. 数据源架构

### 双数据源设计

系统采用**双数据源策略**，根据资产代码特征自动路由：

| 数据源 | 适用资产 | 代码特征 | API 端点 |
|--------|----------|----------|----------|
| **天天基金** | 国内公募基金 | 6位纯数字（如 `020972`） | `http://fundgz.1234567.com.cn/js/{code}.js` |
| **Yahoo Finance** | 美股、ETF、指数 | 字母开头（如 `AAPL`, `^GSPC`） | `https://query1.finance.yahoo.com/v8/finance/chart/{symbol}` |

### 数据源选择逻辑

```
fetch_price(code, category)
        │
        ▼
┌───────────────────────┐
│ code.len() == 6 且    │
│ 全部为数字？           │
└───────────┬───────────┘
            │
      ┌─────┴─────┐
      │ Yes       │ No
      ▼           ▼
┌─────────────┐  ┌─────────────┐
│ 天天基金 API │  │ Yahoo Finance│
│ (优先尝试)   │  │ (唯一选择)   │
└──────┬──────┘  └──────┬──────┘
       │                │
       ▼                ▼
┌─────────────────────────────┐
│ 返回 Option<f64>             │
│ None 表示无数据              │
└─────────────────────────────┘
```

### 特殊场景处理

| 场景 | 处理策略 |
|------|----------|
| QDII 基金 | 天天基金可能无估值，返回 `None`（不回退 Yahoo，因为 QDII 在 Yahoo 无对应代码） |
| 休市时间 | 天天基金返回前一交易日净值；Yahoo 返回最新收盘价 |
| 网络超时 | 10秒超时（天天基金）/ 15秒超时（Yahoo），超时返回 `Err` |

## 4. 核心 API

### fetch_price - 获取单个资产价格

```rust
pub async fn fetch_price(code: &str, category: &str) -> Result<Option<f64>>
```

**参数**：
- `code`: 资产代码
- `category`: 资产类别（`us_stocks` / `cn_stocks` / `counter_cyclical`）

**返回**：
- `Ok(Some(price))`: 成功获取价格
- `Ok(None)`: 数据源无该资产数据
- `Err(e)`: 请求失败或解析错误

**使用示例**：
```rust
// 获取美股价格
let price = fetch_price("AAPL", "us_stocks").await?;
if let Some(p) = price {
    println!("Apple 当前价格: ${:.2}", p);
}

// 获取国内基金估值
let price = fetch_price("020972", "cn_stocks").await?;
```

---

### update_all_prices - 批量更新所有持仓价格

```rust
pub async fn update_all_prices(positions: &[Position]) -> Result<Vec<PriceUpdate>>
```

**参数**：
- `positions`: 持仓列表（来自 `db.list_positions()`）

**返回**：
- `Ok(updates)`: 成功更新的列表（跳过失败的资产）
- `Err(e)`: 严重错误（极少发生）

**处理逻辑**：
```
for position in positions:
    1. 调用 fetch_price()
    2. 成功 → 加入 updates 列表
    3. 无数据 → 打印警告，跳过
    4. 失败 → 打印错误，跳过
返回 updates
```

**特点**：**部分成功语义** - 即使某些资产获取失败，也会返回成功获取的部分。

---

### fetch_full_quote - 获取完整报价信息

```rust
pub async fn fetch_full_quote(symbol: &str) -> Result<StockQuote>
```

**参数**：
- `symbol`: 股票代码（如 `AAPL`, `^GSPC`）

**返回**：
- `Ok(quote)`: 完整报价信息
- `Err(e)`: 获取失败

**使用示例**：
```rust
let quote = fetch_full_quote("^GSPC").await?;
println!("S&P 500: {:.2} ({:+.2}%)", quote.price, quote.change_percent);
```

## 5. 底层实现

### 天天基金 API 解析

天天基金返回 **JSONP 格式**，需手动提取 JSON：

```javascript
// 原始响应
jsonpgz({"fundcode":"020972","name":"国泰纳斯达克","gsz":"1.4395","gszzl":"0.85",...});

// 解析步骤
1. 定位 jsonpgz( 之后、) 之前的内容
2. 使用 serde_json 解析内部 JSON
3. 优先取 gsz（估算净值），fallback 到 dwjz（单位净值）
```

**字段映射**：
| JSON 字段 | 含义 | 优先级 |
|-----------|------|--------|
| `gsz` | 估算净值（盘中实时） | 高 |
| `dwjz` | 单位净值（盘后更新） | 低 |

### Yahoo Finance v8 API 解析

Yahoo 返回结构化 JSON，需遍历嵌套路径：

```json
{
  "chart": {
    "result": [{
      "meta": {
        "regularMarketPrice": 178.72,
        "previousClose": 177.45,
        "shortName": "Apple Inc."
      }
    }]
  }
}
```

**解析路径**：`chart.result[0].meta.regularMarketPrice`

## 6. 错误处理策略

### 错误分类

| 错误类型 | 处理方式 | 用户可见信息 |
|----------|----------|--------------|
| 网络超时 | 返回 `Err` | "请求 XXX API 失败" |
| HTTP 非 200 | 返回 `Ok(None)` | 静默跳过 |
| JSON 解析失败 | 返回 `Err` | "解析响应失败" |
| 价格 ≤ 0 | 返回 `Ok(None)` | 静默跳过 |

### 容错设计

```rust
// update_all_prices 的容错逻辑
for pos in positions {
    match fetch_price(&pos.asset_code, &pos.category).await {
        Ok(Some(price)) => updates.push(...),
        Ok(None) => eprintln!("⚠ 无法获取 {} 价格，已跳过", code),
        Err(e) => eprintln!("⚠ 获取 {} 失败: {}，已跳过", code, e),
    }
}
// 继续返回成功部分，不会因为单个失败而中断
Ok(updates)
```

## 7. HTTP 客户端配置

### 天天基金客户端

```rust
let client = reqwest::Client::builder()
    .timeout(Duration::from_secs(10))
    .build()?;

// 请求头设置
.header("Accept", "*/*")
.header("Referer", "https://fund.eastmoney.com/")
```

### Yahoo Finance 客户端

```rust
let client = reqwest::Client::builder()
    .user_agent("Mozilla/5.0 (Windows NT 10.0; Win64; x64) ...")
    .timeout(Duration::from_secs(15))
    .build()?;

// 请求头设置（模拟浏览器）
.header("Accept", "application/json, text/plain, */*")
.header("Accept-Language", "en-US,en;q=0.9")
.header("Referer", "https://finance.yahoo.com/")
```

> ⚠️ **注意**：Yahoo Finance 需要设置 User-Agent 和 Referer，否则可能返回 403 Forbidden。

## 8. 使用场景

### 场景一：更新所有持仓价格（CLI 命令）

```rust
// main.rs: cmd_update_prices()
async fn cmd_update_prices() -> Result<()> {
    let db = Database::open()?;
    let positions = db.list_positions()?;
    
    let updates = update_all_prices(&positions).await?;
    
    // 更新数据库
    for update in &updates {
        db.update_position_price(&update.asset_code, update.new_price)?;
    }
    
    // 显示更新结果
    print_price_update_table(&updates);
    Ok(())
}
```

### 场景二：生成报告前自动更新

```rust
// main.rs: cmd_report()
async fn cmd_report() -> Result<()> {
    let db = Database::open()?;
    let positions = db.list_positions()?;
    
    // 自动更新价格（静默模式）
    if !positions.is_empty() {
        let updates = update_all_prices(&positions).await?;
        for u in updates {
            db.update_position_price(&u.asset_code, u.new_price)?;
        }
    }
    
    // 继续生成报告...
}
```

### 场景三：获取市场指数

```rust
// market.rs 使用 quote.rs 的能力
pub async fn fetch_market_indices() -> Result<Vec<StockQuote>> {
    let indices = vec!["^GSPC", "^DJI", "^IXIC", "^RUT"];
    let mut quotes = Vec::new();
    for symbol in indices {
        if let Ok(quote) = fetch_full_quote(symbol).await {
            quotes.push(quote);
        }
    }
    Ok(quotes)
}
```

## 10. 限制与注意事项

1. **无代理支持**：当前版本不支持 HTTP 代理，海外用户可能无法访问天天基金
2. **无缓存机制**：每次调用都发起网络请求，频繁调用可能被限流
3. **QDII 基金限制**：国内 QDII 基金在天天基金可能无估值，需要手动更新
4. **交易日限制**：非交易日返回的可能是历史数据，需用户自行判断
