# System Boundary Interface Documentation

This document describes the system's external invocation interfaces, including CLI commands, API endpoints, configuration parameters, and other boundary mechanisms.

## Command Line Interface (CLI)

### init

**Description**: 初始化应用配置文件、数据库和报告目录。如果检测到已有数据，会提示用户确认后再覆盖。

**Source File**: `src/main.rs`

**Arguments**:
- `-f, --force`: 跳过确认提示，强制覆盖已有数据

**Usage Examples**:

```bash
# 普通初始化（如有数据会提示确认）
mns init

# 强制初始化（跳过确认）
mns init --force
```

---

### config

**Description**: 查看或修改配置项，支持动态查询和设置。

**Source File**: `src/main.rs`

**Arguments**:
- `key` (string): optional - 配置项的点路径键名
- `value` (string): optional - 要设置的配置值

**Usage Examples**:

```bash
# 查看所有配置
mns config

# 查看单个配置项
mns config settings.annualized_target_low

# 修改配置项
mns config settings.annualized_target_low 15.0
```

---

### cash

**Description**: 查看当前现金余额。

**Usage Examples**:

```bash
mns cash
```

---

### cash set

**Description**: 设置现金余额为指定值（不能为负）。

**Arguments**:
- `amount` (float): required - 要设置的现金金额

**Usage Examples**:

```bash
mns cash set 100000
```

---

### cash add

**Description**: 向现金余额增加指定金额（年度注资场景）。

**Arguments**:
- `amount` (float): required - 要增加的现金金额

**Usage Examples**:

```bash
mns cash add 50000
```

---

### portfolio

**Description**: 显示当前资产持仓列表，包括成本价、现价、市值、年化收益、绝对收益等。

**Usage Examples**:

```bash
mns portfolio
```

---

### add

**Description**: 添加新资产到投资组合。

**Arguments**:
- `code` (string): required - 资产代码
- `name` (string): required - 资产名称
- `category` (string): required - 资产类别：us_stocks / cn_stocks / counter_cyclical

**Usage Examples**:

```bash
mns add QQQ "纳指100" us_stocks
mns add 512890 "红利低波" cn_stocks
mns add GLD "黄金" counter_cyclical
```

---

### buy

**Description**: 买入指定资产的份额。

**Arguments**:
- `code` (string): required - 资产代码
- `shares` (float): required - 买入份额
- `price` (float): required - 买入价格

**Usage Examples**:

```bash
mns buy QQQ 10.5 380.00
```

---

### sell

**Description**: 卖出指定资产的份额。

**Arguments**:
- `code` (string): required - 资产代码
- `shares` (float): required - 卖出份额
- `price` (float): required - 卖出价格

**Usage Examples**:

```bash
mns sell QQQ 5.0 420.00
```

---

### price

**Description**: 查询或更新资产的当前价格。

**Arguments**:
- `code` (string): required - 资产代码
- `price` (float): optional - 要更新的价格

**Usage Examples**:

```bash
# 查询价格
mns price QQQ

# 更新价格
mns price QQQ 385.00
```

---

### update-prices

**Description**: 自动更新所有持仓资产的当前价格。根据代码格式自动选择数据源：
- 6位数字代码：使用天天基金接口
- 字母代码：使用 Yahoo Finance 接口

**Source File**: `src/quote.rs`

**Usage Examples**:

```bash
mns update-prices
```

**Output Example**:

```
代码        名称          原价格      新价格      来源
QQQ         纳指100       380.00     385.50     Yahoo Finance
512890      红利低波      1.250      1.258      天天基金
```

### sentiment

**Description**: 获取并显示当前恐惧与贪婪指数（使用 finance-query/alternative.me API），并保存至数据库。

**Source File**: `src/sentiment.rs`

**Usage Examples**:

```bash
mns sentiment
```

**Output Example**:

```
正在获取恐贪指数...
恐贪指数: 45 (恐慌)
```

### report

**Description**: 生成综合投资策略报告，包含情绪分析、买卖建议、风险预警、净操作指引，并保存为文件。

**Usage Examples**:

```bash
mns report
```

**Report Output**: `reports/YYYY-MM-DD.txt`

---

### history

**Description**: 显示最近的交易历史记录。

**Arguments**:

```bash
mns history
```

## API Interfaces

### finance-query / alternative.me Fear & Greed Index

**Description**: 通过 finance-query crate 获取恐惧贪婪指数数据。

**Source File**: `src/sentiment.rs`

**Library**: `finance-query = "2"`

**Data Source**: alternative.me API

**Response Format**: 
```json
{
  "value": "45",
  "value_classification": "Fear",
  "timestamp": "1617235200"
}
```

---

### 天天基金接口

**Description**: 用于获取国内基金的实时估值数据。

**Endpoint**: `https://fundgz.1234567.com.cn/js/{code}.js`

**Source File**: `src/quote.rs`

**适用范围**: 6位数字代码（国内基金）

---

### Yahoo Finance API

**Description**: 用于获取美股/ETF的实时报价。

**Endpoint**: `https://query1.finance.yahoo.com/v8/finance/chart/{symbol}`

**Source File**: `src/quote.rs`

**适用范围**: 字母代码（美股/ETF）

## Integration Suggestions

### 自动化定时任务

建议通过系统定时任务（如 cron）定期执行：

```bash
# 每日早上 9 点更新价格并生成报告
0 9 * * * /path/to/mns update-prices && /path/to/mns report
```

### 配置管理集成

建议将 config.toml 配置文件纳入版本控制系统：

```bash
git add .mns/config.toml && git commit -m "update target return to 15%"
```
