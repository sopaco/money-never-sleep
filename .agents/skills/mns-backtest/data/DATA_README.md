## 数据来源说明

### 纳斯达克指数 (nasdaq)
- **来源**: yfinance (^IXIC)
- **说明**: 纳斯达克综合指数，包含在纳斯达克交易所上市的所有股票
- **更新**: 月末收盘价

### 沪深300指数 (hs300)
- **来源**: 手动补充历史数据 (2016-2020) + yfinance (2020-2025)
- **说明**: 中国A股市场最具代表性的蓝筹股指数
- **数据重建**: 2016-2020年数据从公开市场数据重建，2020年后使用 yfinance

### 人民币黄金 (gold_cny)
- **来源**: yfinance (518880.SS 华安黄金ETF)
- **说明**: 作为人民币黄金价格的代理指标
- **注意**: ETF价格不是直接的元/克金价，但走势高度相关

### 恐贪指数 (fgi)
- **来源**: 
  - 2016-2020: GitHub开源数据集 (逐日数据)
  - 2020-2025: 月度采样数据
- **处理**: 取每月最后一个交易日的恐贪指数值
- **区间定义**:
  - 0-24: 极度恐慌 (Extreme Fear)
  - 25-44: 恐慌 (Fear)
  - 45-54: 中性 (Neutral)
  - 55-74: 贪婪 (Greed)
  - 75-100: 极度贪婪 (Extreme Greed)

## 数据使用注意事项

1. **人民币黄金代理**: `gold_cny` 使用黄金ETF价格作为代理，不完全等同于上海黄金交易所的AU9999现货金价，但相关性极高

2. **沪深300早期数据**: 2016-2020年的沪深300数据为手动补充，数据精确度可能略低于 yfinance 直接获取的数据

3. **恐贪指数月度化**: 恐贪指数原始数据为日频，本数据集取每月最后一个有效值作为月末情绪指标

4. **时区差异**: 美股数据使用美国东部时间，A股数据使用北京时间，月末日期可能有1天差异
