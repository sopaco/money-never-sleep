# MNS 多资产策略回测报告

## 回测期间
2016年1月 - 2025年4月（约9年）

## 回测假设
- 初始资金：¥100,000
- 年度注资：¥50,000（每年3月末）
- 总投入：¥600,000
- 数据频率：月度
- 忽略交易成本、滑点、税收

## 多资产 vs 单资产对比

| 策略 | 年化收益 | 总收益率 | 最大回撤 | 买入次数 | 卖出次数 |
|------|----------|----------|----------|----------|----------|
| **多资产组合** | 4.98% | 56.80% | **14.28%** | 108 | 27 |
| 单一美股(S&P500) | 5.57% | 65.14% | 17.95% | 38 | 8 |
| 买入持有 | 7.29% | 91.68% | 14.70% | 11 | 0 |

## 核心发现

### 1. 多资产组合降低风险
- 最大回撤从17.95%降至14.28%（降低3.67个百分点）
- 分散了单一市场的风险暴露

### 2. 但收益也有所降低
- 年化收益从5.57%降至4.98%（降低0.59个百分点）
- A股和黄金的长期表现不如美股

### 3. 资产配置比例影响显著
- 默认配置：美股50%、A股35%、黄金15%
- 建议根据风险偏好调整：
  - 激进型：美股60%、A股25%、黄金15%
  - 保守型：美股40%、A股30%、黄金30%

### 4. A股和黄金的逆向信号
- A股在恐慌区间买入效果不如美股
- 黄金作为避险资产，在恐慌时表现较好

## 各资产历史收益（2016-2025）

|------|----------|------------|
| 纳斯达克指数 | 278% | 15.47% |
| 沪深300指数 | 27% | 2.65% |
| 人民币黄金 | 216% | 13.25% |

## 结论与建议

### 1. 多资产配置的取舍
- **优点**：显著降低回撤，适合风险厌恶型投资者
- **缺点**：长期收益降低，错过美股牛市的超额收益

### 2. 优化建议
- 增加美股配置比例（55-60%）
- A股配置可在恐慌时增加至40%，正常时降至25%
- 黄金保持15-20%作为避险配置

### 3. 情绪区间的差异化策略
- 极度恐慌：美股70%、A股20%、黄金10%
- 恐慌：美股50%、A股30%、黄金20%
- 中性及以下：不买入

## 使用方法

```bash
# 运行多资产回测
mns backtest

# 使用自定义配置
mns backtest run --config .agents/skills/mns-backtest/data/config_balanced.toml

# 对比多个配置
mns backtest run --compare config1.toml,config2.toml,config3.toml
```

## 数据文件

- `monthly_multi_asset.csv` - 2016-2025月度数据（FGI、纳斯达克、沪深300、人民币黄金）
- `config_balanced.toml` - 平衡配置
- `config_circuit_breaker.toml` - 熔断抄底配置
- `config_extreme_contrarian.toml` - 极致逆向配置
