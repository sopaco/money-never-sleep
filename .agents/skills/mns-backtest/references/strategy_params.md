# MNS 策略参数参考

## 核心参数

### 恐贪指数阈值 (THRESHOLDS)

| 区间 | 阈值 | 说明 |
|------|------|------|
| 极度恐慌 (Extreme Fear) | < 25 | 最佳买点 |
| 恐慌 (Fear) | 25-44 | 较好买点 |
| 中性 (Neutral) | 45-55 | 小额试仓 |
| 贪婪 (Greed) | 56-74 | 暂停买入 |
| 极度贪婪 (Extreme Greed) | >= 75 | 启动止盈 |

### 买入比例 (BUY_RATIO)

| 情绪区间 | 投入可用现金比例 |
|----------|----------------|
| 极度恐慌 | 50% |
| 恐慌 | 30% |
| 中性 | 20% |
| 贪婪 | 0% |
| 极度贪婪 | 0% |

### 止盈目标 (ANNUALIZED_TARGET)

- 低线: 10% 年化
- 高线: 15% 年化

### 卖出减仓矩阵 (SELL_RATIO)

根据**情绪区间** + **持仓年化收益** 决定减仓比例：

| 情绪区间 | 年化 >= 15% | 10% <= 年化 < 15% | 年化 < 10% |
|----------|------------|------------------|------------|
| 极度贪婪 (FGI > 75) | 50% | 30% | 20% |
| 贪婪 (FGI 56-74) | 40% | 20% | 0% (持有) |
| 中性 (FGI 45-55) | 30% | 0% (持有) | 0% (持有) |

## 年化收益计算公式

```
年化收益 = (当前价格 / 持仓成本) ^ (365 / 持仓天数) - 1
```

**注意**：上述复利公式适用于浮盈场景。当持仓浮亏时，使用简单年化公式以避免失真：

```
年化收益（浮亏）= (当前价格 / 持仓成本 - 1) / (持仓天数 / 365)
```

**示例**：
- 浮盈：成本100，现价120，持仓365天 → 年化 = (1.2)^1 - 1 = 20%
- 浮亏：成本100，现价80，持仓365天 → 年化 = (0.8 - 1) / 1 = -20%（而非复利公式的 -20%）
- 浮亏短持：成本100，现价95，持仓30天 → 简单年化 ≈ -61%（复利公式会得到约 -46%，但此时通常受 MIN_HOLDING_DAYS 限制不显示年化）

## 资金假设

- 初始资金: ¥100,000
- 每年2月追加: ¥50,000
- 交易成本: 忽略（个人工具，0手续费）

## 回测数据来源

### 恐贪指数 (CNN Fear & Greed Index)
- 2016-2020.09: 逐日数据（GitHub gman4774/Fear_and_Greed_Index）
- 2020.10-2025.04: 月度近似数据（关键时点重建）
- API: `https://production.dataviz.cnn.io/index/fearandgreed/graphdata`
  - 当前值: `fear_and_greed.score`
  - 评级: `fear_and_greed.rating`

### S&P 500 价格
- 月度收盘价近似值
- 作为美股整体表现代理

## 已知回测结论

基于 2016-2025 回测：
- 逆向策略年化 6.93%，买入持有 7.18%
- 逆向策略在熊市更抗跌（2022年: -11.7% vs -13.9%）
- 极度恐慌/恐慌区间买入后3个月胜率 > 80%
- 中性区间买入过频会拉低收益，建议降低中性买入比例至 10% 或 0%
